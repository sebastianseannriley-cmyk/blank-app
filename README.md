
# Car Price Portfolio — Interactive

- Fixed to `data.csv` with tailored columns (Year, Brand, Model/Trim, Price, Mileage, Engine Size, Fuel, Transmission, Condition).
- Highly interactive dashboard:
  - Tabs: Overview · Compare · Drill‑down · Table
  - Clickable legend and bars to filter other charts
  - Brand/Model comparator & synchronized selections
  - Favorites (bookmark vehicles) with CSV export
  - Real‑time ValueScore tuning
  - Download best/worst per year and full table views

## Run
pip install -r requirements.txt
streamlit run app.py
